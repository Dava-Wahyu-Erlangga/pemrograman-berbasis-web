Pertukaran Mahasiswa

Program pertukaran mahasiswa mandiri ini dilaksanakan mulai bulan Agustus hingga Desember 2024, dengan tujuan Universitas Bengkulu, tepatnya di Fakultas Teknik, Program Studi Sistem Informasi. Program ini merupakan bagian dari inisiatif untuk meningkatkan kualitas pendidikan tinggi di Indonesia melalui kolaborasi antarperguruan tinggi. Selama mengikuti program ini, saya berkesempatan untuk merasakan langsung suasana akademik di universitas lain, memperluas relasi, serta memperdalam pemahaman dalam bidang sistem informasi dari sudut pandang yang berbeda.

Melalui program ini, saya belajar untuk lebih mandiri, beradaptasi dengan lingkungan baru, serta menghargai perbedaan dalam keberagaman. Pengalaman yang saya peroleh menjadi bekal berharga dalam pengembangan diri, baik sebagai mahasiswa maupun sebagai calon profesional di bidang teknologi informasi. Saya percaya bahwa pertukaran mahasiswa bukan hanya tentang berpindah tempat belajar, tetapi tentang membuka wawasan dan menciptakan peluang untuk berkembang lebih luas.
